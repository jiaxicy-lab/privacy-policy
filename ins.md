# 隐私权政策

**产品名称：** 灵感库 · Instagram Reels 采集助手  
**最后更新：** 2026 年 6 月 22 日

---

## 1. 概述

本扩展程序（以下简称"本扩展"）尊重并保护用户隐私。本政策说明本扩展收集、使用和存储哪些数据，以及这些数据如何被处理。

---

## 2. 收集的数据

本扩展在用户主动使用时，会处理以下类型的数据：

**2.1 Instagram 内容数据**  
在用户启动自动化采集功能时，本扩展从用户已登录的 Instagram 页面读取公开可见的 Reels 内容，包括：
- 视频标题（文案）及话题标签
- 点赞数、评论数、分享数、播放量
- 视频时长
- 作者用户名及粉丝数
- 封面图片 URL 及视频 URL
- 背景音乐信息

**2.2 用户配置数据**  
用户在扩展设置中手动填写的内容，包括：
- 账号定位描述（Brief）
- 采集过滤规则（关键词、点赞阈值等）
- 第三方 AI 服务的 API Key 及服务地址
- 灵感库推送接口地址（Ingest API）及 API Key

**2.3 本地日志数据**  
扩展在本地生成并保存的操作记录，包括：
- 采集到的 Reels 列表（最多 500 条）
- 灵感库内容列表（最多 500 条）
- 操作日志（最多 300 条）

---

## 3. 数据的使用方式

| 数据类型 | 使用目的 |
|---|---|
| Instagram 内容数据 | 按用户设定的规则进行过滤和展示 |
| 封面图 / 视频关键帧 | 发送至用户指定的 AI 服务进行内容分析和评分 |
| 账号 Brief + Reel 数据 | 构建 AI 分析请求的输入内容 |
| Ingest API 地址及 Key | 将用户选择的内容推送至用户自有的外部系统 |

本扩展**不会**将任何数据用于广告、用户追踪或任何与用户明确授权目的无关的用途。

---

## 4. 数据存储

- **所有数据仅存储在用户本地设备上**，使用 Chrome 浏览器提供的 `chrome.storage.local` API 保存。
- **API Key 存储在本地**，不会被上传至任何由本扩展开发者控制的服务器。
- 用户可随时通过扩展界面中的"清空"功能删除本地存储的采集数据和灵感库数据。
- 卸载本扩展后，Chrome 会自动清除本扩展存储的所有本地数据。

---

## 5. 数据共享与第三方服务

本扩展本身**不收集、不传输、不存储**任何用户数据到开发者服务器。

当用户配置并启用 AI 分析功能时，本扩展会将 Reel 的文本描述和图像数据发送至用户**自行选择并配置**的第三方 AI 服务（如 Anthropic、OpenAI、火山引擎、阿里百炼或自定义服务）。用户需自行阅读并遵守所选 AI 服务提供商的隐私政策及服务条款。

当用户配置灵感库推送功能时，本扩展会将 Reel 元数据发送至用户**自行填写**的 Ingest API 地址。该接口由用户自行掌控，与本扩展开发者无关。

---

## 6. 权限说明

本扩展请求以下 Chrome 权限，均以实现核心功能为限：

| 权限 | 用途 |
|---|---|
| `storage` | 在本地保存用户配置和采集数据 |
| `downloads` | 用户主动触发时将视频文件下载至本地 |
| `sidePanel` | 以侧边栏形式展示扩展主界面 |
| `tabs` | 打开 Instagram 链接及控制采集标签页导航 |
| `scripting` | 向 Instagram 页面注入脚本以读取 Reels 数据 |
| 主机权限（Instagram 及 AI 服务域名） | 读取 Instagram 页面内容并与 AI 服务通信 |

---

## 7. 儿童隐私

本扩展不面向 13 岁以下儿童，也不会故意收集儿童的个人信息。
