隐私权政策 · Privacy Policy

最后更新 / Last Updated：2026-06-17

灵感库 · TikTok 视频采集助手

1. 数据收集

本扩展程序不收集、不存储、不传输任何数据到开发者服务器。

本扩展在运行过程中会处理以下数据，所有数据仅存储在您的本地浏览器（chrome.storage.local）中：

  • TikTok 视频元数据：标题、话题标签、作者信息、播放/点赞/评论等统计数据。仅用于本地筛选和展示。
  • 视频封面图与关键帧截图：用于 AI 分析功能，发送到您自行配置的 AI API 服务，不经过开发者服务器。
  • AI API 密钥：仅存储在本地，仅用于发起 AI 分析请求，开发者无法访问。
  • 自定义 Webhook / 灵感库 API 配置：仅存储在本地，数据直接由您的浏览器推送到您指定的服务。
  • 操作日志与采集配置：本地存储，随时可由您清除。

2. 数据共享

本扩展不向任何第三方共享数据，以下情况除外（均由您主动配置触发）：

  • 您配置了 AI 分析功能时，视频文本及图像数据会发送至您选择的 AI 服务商（如 OpenAI、Anthropic、火山引擎等）。
  • 您配置了灵感库推送功能时，视频数据会推送到您填写的 API 地址。

3. 数据存储与删除

所有数据存储在您本地浏览器的 chrome.storage.local 中。您可以随时在扩展界面内清除全部数据，或通过卸载扩展来完全删除所有本地数据。

4. 权限说明

权限用途
storage在本地保存采集配置、视频数据和日志
tabs控制 TikTok 标签页进行自动化采集
downloads支持视频下载功能
sidePanel在浏览器侧边栏展示扩展界面
notifications采集完成时发送桌面通知
host_permissions: tiktok.com在 TikTok 页面注入脚本以捕获视频数据

5. 未成年人

本扩展不面向 13 岁以下未成年人，也不主动收集任何用户的个人身份信息。

English Version

This extension does not collect, store, or transmit any data to developer servers. All data (TikTok video metadata, AI API keys, webhook configurations, and operation logs) is stored exclusively in your local browser via chrome.storage.local. Data is only sent to third-party AI services or your own API endpoints when you explicitly configure those features. You can delete all data at any time by clearing the extension or uninstalling it.
